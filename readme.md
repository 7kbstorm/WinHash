# WinHash
### Description
LM and NTLM hash generator.

### Built on:
- Net Framework 4.5.2
- Visual Studio 2015
- Tested on Windows 10

### Changes:
- Convert a string to NTLM or LM hash
- Output as PWDump format
- Format text buttons (remove dashes and lowercase)
